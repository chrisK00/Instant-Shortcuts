## Git
 | Title                   | Command      |
 | ----------------------- | ------------ |
 | initialize a local repo | **git init** |

 ## dotnet cli
 | Title                      | Command                         |
 | -------------------------- | ------------------------------- |
 | New app                    | **dotnet new console/classlib** |
 | Run and track code changes | **dotnet watch run**            |
 | run tests                  | **dotnet test**                 |

 ## Open
 | Title                 | Command     |
 | --------------------- | ----------- |
 | Open in VSCode        | **code .**  |
 | Open in file explorer | **start .** |
